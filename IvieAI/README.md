# IvieAI - Friendly Chat Assistant 🤖

IvieAI is an intelligent conversational AI chatbot powered by Hugging Face transformers. It provides friendly, engaging conversations with proper punctuation and complete sentences.

## Features ✨

- 🤖 Intelligent conversations using GPT-2 based model
- 💾 Automatic chat history management
- 📄 Export conversations as plain text or JSON
- 📚 Session management with multiple conversation threads
- 🎨 Clean, responsive web interface
- ⚡ Real-time responses with proper sentence completion

## Tech Stack 🛠️

- **Backend**: Flask, Python 3.9+
- **Frontend**: HTML5, CSS3, JavaScript
- **AI Model**: Hugging Face Transformers (GPT-2)
- **Deployment**: Compatible with Heroku, Railway, or any Python host
